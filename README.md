# Vite & Gourmand

## Présentation

Ce projet  est un site web pour une entreprise de restauration qui propose différents menus pour des événements (Noël, Pâques, etc...).

Le site permet de gérer les menus et de les afficher aux visiteurs.

---

## Fonctionnalités

- Afficher les menus
- Ajouter un menu
- Modifier un menu
- Supprimer un menu

---

## Technologies utilisées

-  HTML
- CSS
- PHP
- MySQL


---

## Installation

- Installer XAMPP
- Mettre le projet dans le dossier htdocs
- Importer la base de données dans phpMyAdmin
- Lancer Apache et MySQL
- Ouvrir le projet dans le navigateur

---

## Base de données

Base : vite_gourmand

Table principale : menus

Champs :
- id
- titre
- description
- prix
- personnes_min
- theme
- regime
